A complete Star Wars-themed CLI tools suite for developers. Bring the power of the Empire (and the wisdom of the Jedi) to your development workflow.

## Features

- 🌑 **Central Router**: One command (`deathstar`) to rule them all.
- ⚔️ **Vader (Git Enforcer)**: Ensures code quality and enforces Imperial Git standards.
- 🧙 **Yoda (Wise Advisor)**: Provides commit message wisdom and developer tips.
- 🤖 **R2-D2 (Diagnostics)**: System health checks and astromech sounds.
- 💥 **Tarkin (Deployments)**: Oversees deployments and branch destruction.
- 📊 **Thrawn (Analytics)**: Deep repository analysis and tactical precision.
- ⚔️ **Lightsaber (Editor)**: Open files with glowing lightsaber effects.
- 🚀 **Hyperdrive (Navigation)**: Hyperspace jumps between directories.
- 🎯 **Stormtrooper (Finder)**: Fuzzy file finding with Imperial precision.
- 👑 **Palpatine (AI Enhancer)**: Unlimited power for your AI prompts.

## Installation

```bash
# Clone or navigate to the DEATHSTAR directory
cd DEATHSTAR

# Make the main script executable
chmod +x deathstar commands/*.sh

# (Optional) Add to your PATH
export PATH="$PATH:$(pwd)"
```

## Quick Start

```bash
# Show system status
./deathstar status

# Review your code before committing
./deathstar vader review

# Get some wisdom
./deathstar yoda wisdom
```
