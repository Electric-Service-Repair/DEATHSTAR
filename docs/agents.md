# Agents of the Death Star

A comprehensive guide to the characters and entities that power the DEATHSTAR CLI suite.

## 🌑 Imperial High Command

These agents enforce the Empire's standards and oversee the operations of the Death Star.

### **Darth Vader (The Git Enforcer)**
*Command: `deathstar vader`*
The ultimate enforcer of code quality and repository standards.
- **`push`**: A protected git push that warns about sensitive branches and uncommitted changes.
- **`review`**: Scans the codebase for TODOs, `console.log` statements, and long functions that defy Imperial efficiency.

### **Grand Moff Tarkin (The Overseer)**
*Command: `deathstar tarkin`*
Responsible for the deployment and destruction of planetary targets (and branches).
- **`deploy`**: Simulates a deployment to a specified branch or environment.
- **`planet`**: Selects a target planet (deployment environment) for Imperial focus.

### **Grand Admiral Thrawn (The Tactician)**
*Command: `deathstar thrawn`*
Provides deep analytical insights into the repository's health and structure.
- **`repo`**: Performs a full repository audit, calculating a tactical health score based on activity and structure.

### **Emperor Palpatine (The Architect)**
*Command: `deathstar palpatine`*
Harnesses "unlimited power" through AI prompt enhancement and templates.
- **`palpatine`**: Generates and refines AI prompts using Imperial templates for maximum influence.

### **Stormtroopers (The Legion)**
*Command: `deathstar stormtrooper`*
The rank-and-file of the Imperial search forces.
- **`find`**: Fuzzy file searching across the entire repository.
- **`aim`**: Interactive file targeting for precision operations.

## ✨ Rebel Contacts & Droids

These entities provide wisdom, diagnostics, and essential utility services.

### **Master Yoda (The Wise Advisor)**
*Command: `deathstar yoda`*
Offers guidance and wisdom from the light side of the Force.
- **`commit`**: Suggests clear, concise commit messages that reflect the true intent of the changes.
- **`wisdom`**: Provides random developer wisdom and quotes to guide your path.

### **R2-D2 (The Astromech)**
*Command: `deathstar r2d2`*
Handles system diagnostics and provides essential technical support.
- **`scan`**: Runs comprehensive system diagnostics (disk, memory, git health).
- **`beep`**: Plays a classic astromech audio cue (requires system speaker support).

---

## 🛠️ Imperial Technology

The machinery and equipment used by all agents of the Empire.

### **Hyperdrive (Navigation)**
*Command: `deathstar hyperdrive`*
Allows for rapid transit between repository sectors.
- **`jump`**: Navigates to a specific directory with a hyperspace animation.
- **`coords`**: Displays your current coordinates in the repository.

### **Lightsaber (The Weapon)**
*Command: `deathstar lightsaber`*
An elegant tool for a more civilized age of code editing.
- **`edit`**: Opens a file in your preferred editor with glowing visual effects.
