# 👑 Palpatine (AI Enhancer)

Emperor Palpatine grants you "unlimited power" for your AI interactions, helping you craft more effective prompts.

## Commands

### `prompt`
**Usage:** `deathstar palpatine "[text]"`
Takes your input and wraps it in a template designed to get the best results from AI models.

### `templates`
**Usage:** `deathstar palpatine templates`
Lists all available prompt templates (e.g., power, detailed, creative, code, explain).

### `help`
**Usage:** `deathstar palpatine help`
Displays the help menu for the Palpatine subcommand.
