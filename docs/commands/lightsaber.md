# ⚔️ Lightsaber (Editor Launcher)

The Lightsaber tool allows you to open your favorite editor with a touch of elegance and power.

## Commands

### `edit`
**Usage:** `deathstar lightsaber edit [file] [options]`
Opens the specified file in your default editor.
- `--color=[color]`: Specify the lightsaber glow color (green, blue, red, purple, yellow, cyan).

### `colors`
**Usage:** `deathstar lightsaber colors`
Lists all available lightsaber colors.

### `help`
**Usage:** `deathstar lightsaber help`
Displays the help menu for the Lightsaber subcommand.
