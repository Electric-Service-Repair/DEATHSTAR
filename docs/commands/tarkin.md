# 💥 Tarkin (Deployment Orchestrator)

Grand Moff Tarkin oversees the deployment of the Empire's digital assets. Use these commands to manage your releases and eliminate rebel branches.

## Commands

### `deploy`
**Usage:** `deathstar tarkin deploy [environment]`
Simulates a deployment process. Watch the superlaser charge as your code is "fired" into the specified environment.

### `planet`
**Usage:** `deathstar tarkin planet [branch-name]`
Force-deletes a branch with dramatic flair. Use this to permanently eliminate redundant or rebel branches from your local repository.

### `help`
**Usage:** `deathstar tarkin help`
Displays the help menu for the Tarkin subcommand.
