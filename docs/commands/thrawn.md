# 📊 Thrawn (Repo Analytics)

Grand Admiral Thrawn uses tactical brilliance to analyze your repository's structure and history.

## Commands

### `repo`
**Usage:** `deathstar thrawn repo`
Generates a detailed report on your repository, including:
- Commit frequency and patterns
- Contributor statistics
- File type distribution
- Code health and complexity scores

### `help`
**Usage:** `deathstar thrawn help`
Displays the help menu for the Thrawn subcommand.
