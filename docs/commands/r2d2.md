# 🤖 R2-D2 (System Diagnostics)

R2-D2 is your faithful astromech droid, providing essential diagnostics and system health information.

## Commands

### `scan`
**Usage:** `deathstar r2d2 scan`
Performs a comprehensive system scan, including:
- Disk space usage
- Memory availability
- Git repository status
- Network connectivity (simulated)

### `beep`
**Usage:** `deathstar r2d2 beep`
Plays iconic R2-D2 astromech sounds to signal completion of tasks or just for fun.

### `help`
**Usage:** `deathstar r2d2 help`
Displays the help menu for the R2-D2 subcommand.
