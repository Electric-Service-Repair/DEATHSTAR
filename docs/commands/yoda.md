# 🧙 Yoda (Wise Advisor)

Master Yoda provides guidance to developers, helping them write better code and maintain a healthy mental state.

## Commands

### `commit`
**Usage:** `deathstar yoda commit`
Analyzes your current git diff and suggests a commit message written in Yoda's unique dialect.

### `wisdom`
**Usage:** `deathstar yoda wisdom`
Provides random developer tips, quotes, and words of encouragement to keep your spirits high during long coding sessions.

### `help`
**Usage:** `deathstar yoda help`
Displays the help menu for the Yoda subcommand.
