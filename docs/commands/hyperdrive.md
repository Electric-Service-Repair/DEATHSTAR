# 🚀 Hyperdrive (Fast Navigation)

The Hyperdrive system allows you to jump across your file system at lightspeed.

## Commands

### `jump`
**Usage:** `deathstar hyperdrive jump [directory]`
Navigates to the specified directory with a hyperspace animation.

### `coords`
**Usage:** `deathstar hyperdrive coords`
Shows your current coordinates (directory) and identifies nearby systems (subdirectories) for navigation.

### `help`
**Usage:** `deathstar hyperdrive help`
Displays the help menu for the Hyperdrive subcommand.
