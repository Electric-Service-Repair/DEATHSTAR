# 🎯 Stormtrooper (Fuzzy File Finder)

Stormtroopers are deployed to find specific files within your vast Imperial archives.

## Commands

### `find`
**Usage:** `deathstar stormtrooper find "[pattern]"`
Performs a fuzzy search for files matching the given pattern.

### `aim`
**Usage:** `deathstar stormtrooper aim "[filename]"`
Attempts a precise match for a specific file (accuracy may vary, as per Stormtrooper standards).

### `help`
**Usage:** `deathstar stormtrooper help`
Displays the help menu for the Stormtrooper subcommand.
